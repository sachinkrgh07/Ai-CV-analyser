# 🤖 AI CV Analyzer

AI-powered Resume & CV Analysis Web App built using Python and Streamlit.

## 🚀 Live Demo

https://ai-cv-analyser-created-by-sachin.streamlit.app/

---

## 📌 Features

- Resume Analysis
- ATS Score Checking
- Skill Detection
- Keyword Analysis
- AI Suggestions
- PDF Resume Upload
- Streamlit Web Interface

---

## 🛠️ Technologies Used

- Python
- Streamlit
- NLP
- Machine Learning
- PyPDF2
- Scikit-learn

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
